# Wemayla GitHub Pages Site (Static)

## Quick start
1) Create a GitHub repository (e.g., `wemayla-site`).
2) Upload all files/folders in this zip to the repository root.
3) GitHub: Settings → Pages → Source: `main` / `(root)` → Save.
4) Visit the Pages URL to confirm it works.

## Custom domain (www.wemayla.com)
1) Settings → Pages → Custom domain: `www.wemayla.com` → Save.
2) In your domain DNS:
   - A records (apex/root):
     185.199.108.153
     185.199.109.153
     185.199.110.153
     185.199.111.153
   - CNAME:
     www  ->  <YOUR_GITHUB_USERNAME>.github.io
3) Back in GitHub Pages, enable “Enforce HTTPS” (after it becomes available).
